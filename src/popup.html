Koyfin UI Translator

Static interface labels are translated with a fully offline dictionary. Changes apply immediately on open Koyfin tabs.

Get a free key in the Google Cloud Console: create a project, enable the Cloud Translation API, then Credentials → Create API key. Free tier: 500,000 characters/month.

Typical usage: browsing news costs a few thousand characters per page; one earnings-call transcript is ~40,000. Beyond the free tier Google charges ~$20 per million characters — set a budget alert (e.g. $1) in Google Cloud Billing to avoid surprises.
